# DS075. Merge sort
merge sort를 이용하여 오름차순으로 정렬한 것을 출력하십시오.
단, DEBUG 매크로를 정의하여 컴파일 하면 정렬되는 모습이 출력되고,
매크로가 없는 경우에는 Sorting 결과만 출력 되도록 작성하라.

---

input : 첫 번째 줄은 수열의 개수, 두 번째 줄은 수열
```
6  
12 11 13 5 7 6
```
output :
```
[12] [11] [13] [5] [7] [6]  
[11] [12] [13] [5] [7] [6]  
[11] [12] [13] [5] [7] [6]  
[11] [12] [13] [5] [7] [6]  
[11] [12] [13] [5] [6] [7]  
[5] [6] [7] [11] [12] [13]
```
